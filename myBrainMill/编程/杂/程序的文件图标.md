# 在vs项目中给开发的程序增加图标
1. 项目右键单击，选择Add，选择recources，选择icon，导入一个已有的icon文件，分辨率最好`256*256`
2. 发现文件夹中新建一个rc文件，一个aps文件，一个resource.h文件
3. 在vs的项目结构中导入了导入了icon文件，rc文件，resource.h文件
[如何：创建资源 (C++) | Microsoft Learn](https://learn.microsoft.com/zh-cn/cpp/windows/how-to-create-a-resource-script-file?view=msvc-170)

vs的图标字典说明，有些图标的含义是固定的，不要自己给这些图标新的含义
[Visual Studio 的图像和图标 - Visual Studio (Windows) | Microsoft Learn](https://learn.microsoft.com/zh-cn/visualstudio/extensibility/ux-guidelines/images-and-icons-for-visual-studio?view=vs-2022#BKMK_IconDesign)
