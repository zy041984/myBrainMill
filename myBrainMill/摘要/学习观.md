## 1.5文理
知识不是信息，是映射，即有输入有输出有函数关系。注重怎么从输入得出输出，有时候输入到输出变体少，就可以简化为记忆。有时候输入到输出的变体很多，就要泛化出规律
例如，太阳每天从东方升起，输入是哪天，输出是太阳升起的方向。给出任意一天，就可以得出太阳升起的方向为东，这种情况可以简化为记忆。
例如，车祸中如何判断双方责任，输入是车祸具体情况，输出是双方责任。给出任意一个车祸，要具体情况具体分析双方的责任，这种情况就要按照某种方法某种过往判例得到输出

学习不是记忆，而是学会从已知求出未知，即用多个例子来泛化某个知识点的普遍规律

遗忘恰恰是大脑的生理特征，想要记住知识，就要在多个场合多个场景学习同一知识，这样大脑就不会把单一场景中记住的事情从记忆中优化掉

## 08 感知与观念
从器官中感知到的并非世界的真相，而是由自然演化出来的视觉模型对神经信号的加工
即感知源于模型
模型从哪里来，每个人自己构建的
如何构建？学习，自己从零创建，以及从前人的成果中选择一部分与自己已有的融合
模型的用处，人类有了模型，就有了信心，觉得自己使用模型就可以从感官的输入得出对世界的认识，对某种事件的评价
模型的几个维度 
- 生理方面，依靠人类的生理特征形成，个人无法改变。你不可能像鹰眼一样从高空依然能清晰地看到地面
- 认知方面，用科学体系作为评价标准，从客观事实中找到认识世界的规律
- 观念方面，评价标准很多，不只有科学，伦理，道德，立场都可以作为评价某事的标准
总之有了模型，就像一个人建立了认识论，人就靠着模型来对万物万事形成自己的认识。
所以每个人自己负责自己的模型，莫轻信他人

## 10 信息
当某事对某人而言有多种可能性时
- 具体是哪种情况的不确定性叫做信息熵entropy，例如一道选择题的答案到底是ABCD的哪个
- 能消除人对事的不确定性的事物叫消息info
- 不能消除人对事的不确定性的事物叫噪声noise
- 概率就是某事为某个情况的可能性
从上面看出信息是一个物理量，有单位bit。
每个消息必须有个前提，某人对某事
信息与信息熵的关系，当观察者对某事一无所知时，信息熵与信息，数量相等，意义相反

例子
掷一个硬币，哪面向上？这件事对于某人肯定是不确定的，所以结果是哪面向上就是一个消息。由于正面向上和反面向上的概率是相等的，所以这个消息就是等概率独立事件
我们把这个等概率独立事件的消息定义为1bit
将来我们要考虑的事件和消息与这种情况相比，就可以计算出那件事情的消息具体是几个bit
有一个具体的计算公式
`sigma(全部可能事件，每个事件概率*log(2,1/每个事件概率))`

例如一个选择题四个选项，我不知道该选哪个，各选项就是四个等概率独立事件，每个选项的概率是25%，总消息就是`25%*log(2,1/25%)+25%*log(2,1/25%)+25%*log(2,1/25%)+25%*log(2,1/25%)=log(2,1/25%)=log(2,4)=2bit`

如果我知道50%的概率选C，则ABD的概率都是1/6，总消息就是`1/6*log(2,1/1/6)+1/6*log(2,1/1/6)+50%*log(2,1/50%)+1/6*log(2,1/1/6)=1/2*log(2,1/1/6)+50%*log(2,1/50%)=1/2*log(2,6)+50%*log(2,2)=1/2*log(2,6)+1/2=1.79bit`

## 11为何人不能永生
| 概念                          | 例子                                                                                                         |
| --------------------------- | ---------------------------------------------------------------------------------------------------------- |
| 宏观态 一个孤立系统有哪些可能形态           | 一间屋子的物品摆放可能有很多种，书在书架上，碗在碗橱中，笔在笔筒中，这是一个宏观态  整齐 书在书架上，碗在笔筒中，笔在碗橱中，这是一个宏观态  一般乱 书在笔筒中，碗在笔筒中，笔在碗橱中，这是一个宏观态  最乱 |
| 微观态 每个可能的形态的具体可能性           | 第二个宏观态中，每本书在书架上的具体位置不同，每种摆放形式就是一种微观态                                                                       |
| 可以定性地说，微观态越多，那个宏观态的热力学熵越大   | 最乱那种宏观态，每个物体没放到正确位置的可能形式最大，即微观态最多                                                                          |
| 一个系统的宏观态是可以改变的              | 那个屋子可以从第一个宏观态整齐变化到第二个宏观态一般乱再变化到第三个宏观态最乱                                                                    |
| 而且根据热力学第二定律会自发滑向热力学熵最大的宏观态， | 物品随便放，那就会变成最乱                                                                                              |
| 无法自行逆转。只有输入能量才能让系统变化到其它的宏观态 | 必须有人故意收拾房间，才会变成整洁这个宏观态                                                                                     |

在此基础上薛定谔提出了生命的本质，生命以负熵为食
揭示了生命与非生命的区别
不是物质和能量交换
不是遗传物质
不是信息
而是生命必须摄入能量，才能维持自身的宏观态
所以为啥人不能保存永生，假设人的所有粒子看成一个系统，活着这个宏观态肯定不是这些粒子的最大熵的宏观态，如果没有能量，这些粒子就会自发变向最大熵的宏观态，也就是粒子无序分布的微观态最多的那种宏观态

推广到整个宇宙，热寂heatdeath，就是宇宙的归宿，宇宙在这个宏观态上热力学熵最大，不可能变化到另外一种宏观态

信息与能量也可以有数量上的计算方法，信息熵与热力学熵有转换关系

麦克斯韦妖的思想实验
孤立系统，为什么不能小妖把高温粒子与低温粒子放在盒子两侧来创造温差，从而可以对外输出能量。

因为

观察     获取    消除       消除    摄入
    相当于  相当于     等价于    需要
分子运动  信息    信息熵     热力学熵 能量

所以
小妖需要摄入能量才能把高温粒子与低温粒子分开，虽然后面系统又能输出能量，但这只不过是小妖摄入的能量
